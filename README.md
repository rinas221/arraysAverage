Имеется массив чисел. Необходимо написать функцию, которая будет находить среднее арифметическое всех чисел этого массива и возвращать его. Вызвать функцию для нижеперечисленных массивов, результаты работы функции выводить в консоль.

Массивы:

[1, 5, 12, 4, 3]

[2, 3, 4, 5, 6, 7, 8, 9, 10]

[3, 10, 17]

Результат в консоли:

5
6
10
